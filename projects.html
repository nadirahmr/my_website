Machine Learning Classification on Lung Cancer Data

In this project, I applied machine learning techniques to classify lung cancer survey data. I handled data preprocessing tasks such as managing missing values, removing noise and duplicates, and refining attributes. I implemented four classification algorithms—Decision Tree (DT), Support Vector Machine (SVM), K-Nearest Neighbors (KNN), and Naive Bayes (NB)—and compared their performance. The results were visualized using a bar chart, leading to a final discussion and conclusion on the effectiveness of each model. Tools: Python, Jupyter Notebook.

OpenGL Graphics System Development

In this group project, we developed a computer graphics system using OpenGL in Microsoft Visual Studio 2022. The project involved setting up the development environment, creating graphics projects, and making multiple modifications to enhance the system's features.
